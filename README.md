Ansible role for python3-pip
==================================

Installs pip3 module

Requirements
------------

Python3 must be installed in order for this role to work

Dependencies
------------

None

Example Playbook
----------------

```yaml
- hosts: all
  roles:
    - role: ansible-role-python3-pip
```
